# WhackAMole2D
 Unityを使ったモグラ叩きゲームです

# 実装内容
- ゲーム画面、ホーム画面、スコア表示画面をそれぞれ3つのシーンに分けて制作しました
- ゲーム画面において、モグラは0.5秒かけて出現→3秒維持→0.5秒かけて消滅するようにアニメーションを設定しました
- 制限時間は30秒です
- 3シーンとも共通のBGM、、またハンマーをクリックした際のSEを設定しました
